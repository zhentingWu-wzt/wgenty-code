# Archived: Built-in CodeGraph Benchmarks

These scripts benchmarked the built-in tree-sitter/SQLite CodeGraph indexer
that was removed in the MCP migration (2026-07-10).

The project now uses a third-party `codegraph` CLI via MCP stdio transport.
These scripts are preserved for historical reference only and will not run
as-is because `wgenty-code codegraph index` no longer exists.
